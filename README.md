# Weather App

This is a **Weather App** built using EJS (Embedded JavaScript). The app allows users to check current weather conditions and forecasts for various locations. It provides real-time weather updates, including temperature, humidity, wind speed, and more.

### Demo

You can view a live demo of the project [here](https://weather-app-nlx0.onrender.com/) .

## Features

### User
- View current weather conditions
- Weather forecast for top 50 countries
- Check weather forecasts for multiple days
- Search for weather by city or location
- View weather details including temperature, humidity, wind speed, and more

## Tech Stack
- **Frontend**: EJS, Bootstrap (or any CSS framework you used)
- **Backend**: Node.js, Express.js
- **Weather API**: [OpenWeatherMap](https://openweathermap.org/api)
